================================================================================
                        INREACH - FRONTEND PROCESS & UI/UX GUIDE
================================================================================

This document outlines the end-to-end frontend architecture, user flows, dynamic states,
interactive screen layouts, button actions, and conditional redirection systems 
powering the InReach Android Application (built in Kotlin with Jetpack Compose).

--------------------------------------------------------------------------------
1. APPLICATION ENTRY & SPLASH LIFECYCLE
--------------------------------------------------------------------------------
Upon launching the application, the user enters a highly polished multi-stage splash experience:

A. Stage 1: Hand Loading Animation (0 - 5 Seconds)
   • Visual Concept: Recreates a famous pure-CSS hand finger-tap loading loop
     within Compose Canvas and Layout.
   • Elements: 4 fingers and 1 thumb in white with cobalt blue background elements.
   • Animation: Fingers cycle through sequential elastic vertical compressions and
     nail-bed displacements with custom sine-wave eased animations.
   • Redirection Trigger: After exactly 5.0 seconds (managed via Kotlin coroutine delays), 
     the screen background color smoothly transitions from loading blue (#4492F4) 
     to deep cosmic navy (#0B132B) while transitioning to Stage 2.

B. Stage 2: InReach Portal Logo Pop-up (5 - 8 Seconds)
   • Visual Concept: An elastic spring-scale and alpha fade-in of the official InReach logo.
   • Elements:
     - Centerpiece: A high-fidelity, vector-drawn tech globe backdrop inside Jetpack Canvas,
       complete with coordinate rings and vertical meridians, superimposed under a crisp
       white portal archway and an ascending stylized white human silhouette reaching upward.
     - Brand Typography: Large bold minimalist title rendering "In" (White) and "Reach" (Cobalt blue).
     - Subheading: "PORTAL OF SECURE OPPORTUNITIES" in light-spaced monospace.
   • Redirection Trigger: After 3.0 seconds, the splash screen transitions to "done", 
     redirecting the user to either the Authentication screen (if unauthorized) or Main App.

--------------------------------------------------------------------------------
2. AUTHENTICATION & INITIALIZATION FLOWS (AuthLayout)
--------------------------------------------------------------------------------
If no active session exists, the splash screen routes the user to the polished Auth Screen:

• Fields:
  - Corporate/Passport Email Input Field
  - Password Input Field
• Dynamic Controls:
  - Login Tab: Standard authentication access. Includes a prominent "Login to Portal" button
    which verifies credentials and sets `isLoggedIn = true`.
  - Register Tab: Expandable screen allowing new nodes to configure their global Passport.
• Redirection Destinations:
  - Succeeding on "Login" or completing "Register" instantly triggers a state recomposition
    granting entry to the Unified Dashboard Interface.

--------------------------------------------------------------------------------
3. THE UNIFIED NAVIGATION BAR
--------------------------------------------------------------------------------
The main application uses a dual Navigation layout depending on screen width (Mobile vs. Tablet support):
• Bottom Navigation Bar (Compact screens) & Navigation Rail (Expanded/Tablet screens)
• Tabs & Redirection destinations:
  1. 👤 Profile Tab       (Manage user's personal Passport & details)
  2. 🌍 Passports         (Discover other registered network nodes)
  3. 📥 Inbox             (Handle inbound queries & messages)
  4. 💼 Workspace         (Real-time tools: whiteboard, docs, tasks, meetings, milestones)
  5. 📊 Analytics         (Graphing of network index & node activity)

--------------------------------------------------------------------------------
4. DETAILED SCREEN-BY-SCREEN HIGHLIGHTS & BUTTON INTERACTIVE STATES
--------------------------------------------------------------------------------

------------------------- TAB 1: PROFILE PASSPORT -------------------------
The center of identity. Shows the user how their profile appears to other remote query nodes:
• Interactive Features:
  - Edit Profile Toggle: Click "Edit Passport Info" to open form inputs for name, bio, 
    and customizable skills tags list.
  - Verification Gauge: Visual bar charting "Profile Strength and Identity Index".
  - Mock Resume/Bio Uploader:
    * Button: "Upload Digital PDF/JSON Resume"
    * Interaction: Triggers a file read simulation, loading skills, educational metrics, 
      and network credibility multipliers into the local state.
    * Feedback: Toggles a success feedback toast upon completion.

------------------------- TAB 2: PASSPORTS (BROWSER) -------------------------
A panoramic roster representing all nodes available in the network:
• Interactive Card Buttons:
  - "Query Node Passport": Opens detailed modal showing the node's bio, trust ratings, and tags.
  - "Direct Message Query" (Button): Opens an inline communication portal allowing the user 
    to send a direct message proposal pitch.
  - "Request Warm Introduction" (Button): If direct path is closed, this initiates a routing proposal.
    * Opens bottom dialog sheet to compose introduction text.
    * Choose Mutual Contact: Selects which registered mutual intermediate node acts as the 
      introductory pipeline.
    * Button "Route Introduction Plan Request": Registers proposal in ViewModel, notifies
      the system, and shows a Toast.

------------------------- TAB 3: CENTRAL DIRECTORY INBOX -------------------------
Your localized processing mailbox for connection authorizations:
• Interactive Elements:
  - Queue Roster: Chronological queue of direct messages and collaboration inquiries.
  - Message Action Buttons:
    * "Acknowledge / Read": Flags a message status as READ.
    * "Accept/Reply Connection": Establishes a permanent node link between the two parties.
    * "Decline / Archive": Removes or archives the proposal.

------------------------- TAB 4: WORKSPACE PLATFORM -------------------------
A modular collaborative toolbar panel suite holding 5 localized productivity systems:
• Tab Sub-Selectors inside the Workspace:
  1. Whiteboard: Freeform SVG sketch space. Press "Draw Line", "Draw Rect", or "Clear Canvas" 
     buttons to paint custom vectors onto a responsive shared canvas.
  2. DocEditor: Markdown draft board. Features text area for technical proposals and a 
     "Save Proposal Draft" button.
  3. TaskBoard: Kanban checklist board. "Add New Project Card" floating action button (FAB) 
     creates item cards containing statuses (To-Do, Active, Complete) with task swipe buttons.
  4. Meetings: Dynamic virtual audio timeline. "Join Virtual Call Room" button establishes
     the status of a streaming visual group connection.
  5. Milestones: Roadmaps visualization tracker. "Mark Phase Completed" checkboxes trigger
     visual percentage bars to progress forward dynamically.

------------------------- TAB 5: ANALYTICS -------------------------
Calculates and plots network performance index ratios:
• Features:
  - Rendered bar charts detailing connection volumes.
  - Node density gauges.
  - Visual circular meters indicating Overall Network Quality and Trust.
